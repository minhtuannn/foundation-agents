How long does it take to make progress in Kaggle?
I am one of the first users of Kaggle - more precisely, user number 650.
After registering about 15 years ago, life happened and I never managed to participate much. Only recently, this year, I was able to complete my registration and join some competitions.
Am I the slowest Kaggle participant ever? To verify this, let's compute how long does it take for users to achieve a new rank, and see how the distribution looks like (spoiler alert: I really am!)
Unfortunately we are only able to compute the time it has taken for a user to get to their current rank since they registered. It is not possible to know how long it took them to go between one rank and the others, because the data is not saved.

log_data <- time_to_current_rank %>%
  filter(PerformanceTierFactor == your_tier) %>%
  mutate(
    delay_days = as.numeric(time_to_current_tier),
    log_delay = log1p(delay_days)  # log1p handles 0 safely
  )
my_log_delay <- log_data %>%
  filter(UserId == 650) %>%
  pull(log_delay)
log_mean <- mean(log_data$log_delay, na.rm = TRUE)
log_sd   <- sd(log_data$log_delay, na.rm = TRUE)

z_score <- (my_log_delay - log_mean) / log_sd
p_value <- pnorm(z_score, lower.tail = FALSE)  # one-sided upper tail

cat("Log-transformed Z-score:", round(z_score, 2), "\n")
cat("Parametric one-sided p-value:", signif(p_value, 4), "\n")
